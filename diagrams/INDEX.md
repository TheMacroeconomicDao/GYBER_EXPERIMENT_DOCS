## Format Guide

| Format | Best For | File Size | Quality |
|--------|----------|-----------|---------|
| `.svg` | Web, docs, scaling | Small | Vector (infinite) |
| `.png` | Print, presentations | Large | 300 DPI raster |
| `.mmd` | Editing, version control | Tiny | Source code |
